# 11 — Những lỗi thường gặp

- **Đường dẫn game chưa hợp lệ**: thiếu file bắt buộc → chọn đúng thư mục game, hoặc xác minh file qua Steam.
- **Không có quyền ghi**: đóng game, tắt phần mềm đang khóa file, thử lại.
- **Chữ ký gói không hợp lệ**: gói bị sửa/không phải bản chính thức → tải lại từ nguồn tin cậy.
- **Không đủ dung lượng**: giải phóng ổ đĩa game.
- **Gỡ nhưng không có backup**: dùng Steam → *Xác minh tính toàn vẹn file*.
