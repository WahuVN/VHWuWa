# 01 — Cách tìm thư mục game

1. Mở **Trang chủ** trong VHWuWa.
2. Bấm **🔍 Tự tìm game** — ứng dụng dò ở Steam, Registry và các ổ đĩa.
3. Nếu không thấy, bấm **📁 Chọn thư mục** và trỏ tới thư mục **cài đặt game** (nơi chứa file thực thi).
4. Trạng thái sẽ hiển thị **Hợp lệ** nếu đủ file bắt buộc (đọc từ `Config/game.json`).

Mẹo:
- Trên Steam: bấm chuột phải game → *Quản lý* → *Duyệt file cục bộ*.
- Đường dẫn có thể nằm ở ổ khác ổ C, trong `Program Files`, hoặc có ký tự tiếng Việt — đều được hỗ trợ.
