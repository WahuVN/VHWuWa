# 02 — Cách cài Việt hóa

1. Chọn đúng **thư mục game** ở Trang chủ (trạng thái phải là *Hợp lệ*).
2. Vào trang **Cài Việt hóa** → **📦 Chọn gói .vhwpack**.
3. Xem phần tóm tắt & danh sách file sẽ được cài.
4. Bấm **⬇ Cài đặt**. Ứng dụng sẽ:
   - Kiểm tra chữ ký & SHA-256 của gói.
   - Tạo **bản sao lưu** file gốc.
   - Cài file và kiểm tra lại.
5. Theo dõi thanh tiến trình; có thể **✖ Hủy** an toàn (sẽ tự khôi phục).

> Nếu game đang chạy, hãy đóng game trước khi cài.
