# 07 — Cách chỉnh đồ họa

1. Trang **Đồ họa** → chọn **preset** (Mặc định / Máy yếu / Cân bằng / Chất lượng cao) hoặc chỉnh từng mục.
2. Bấm **✔ Áp dụng tùy chỉnh** hoặc chọn preset để ghi vào file cấu hình game (có backup).
3. **📄 Mở file cấu hình** để xem trực tiếp.

> Nếu game chưa được cấu hình đồ họa (`Config/graphics.json`), trang sẽ báo: *Tính năng này chưa được cấu hình cho game hiện tại.*
