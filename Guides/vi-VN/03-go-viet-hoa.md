# 03 — Cách gỡ Việt hóa

1. Trang chủ → **🗑 Gỡ Việt hóa**.
2. Ứng dụng đọc trạng thái cài đặt, **khôi phục file gốc từ bản sao lưu**, và xóa các file do bản Việt hóa tạo.
3. Nếu **không tìm thấy bản sao lưu**, ứng dụng sẽ KHÔNG tự xóa file lạ để tránh hỏng game — hãy dùng Steam → *Xác minh tính toàn vẹn file*.
